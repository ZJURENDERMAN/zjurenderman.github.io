Vulkan Ray Tracing Project

基于Vulkan的光线追踪项目

Duration: Jun. 2019 - Jun. 2021

起止时间：2019.6 - 2021. 6

  1. We developed the ray tracing algorithm using Vulkan features without RTX on Kirin 990 GPU. 我们在移动端麒麟990GPU上使用Vulkan特性实现了光线追踪算法。
  2. We implemented real-time ray tracing denoisers such as BMFR and SVGF. 我们实现了实时光线追踪降噪算法，例如BMFR和SVGF。
  3. We supported cross-platform rendering on both PC and mobile. 我们在PC端和移动端实现了跨平台的渲染。

OpenGL ES Ray Tracing Project

基于OpenGL ES的光线追踪项目

Duration: Jul. 2021 - Jun. 2022

起止时间：2021.7-2022.6

  1. We developed the ray tracing algorithm using OpenGL ES shaders on Snapdragon 855 GPU. 我们在移动端骁龙855GPU上使用OpenGL ES着色器实现了光线追踪算法。
  2. We supported vertex animation and real-time HLBVH construction on compute shader. 我们支持顶点动画，并且支持计算着色器的HLBVH实时构建
  3. We implemented point light, parallel light and environment light on noise-free shading. 我们支持点光源、平行光以及环境光的无噪声着色。

WebGPU Ray Tracing Project

基于WebGPU的光线追踪项目

Duration: Apr. 2022 - Jun. 2022

起止时间：2022.4-2022.6

  1. We developed the ray tracing algorithm on WebGPU using Vulkan backend with JavaScript. 我们基于WebGPU的Vulkan后端，使用JavaScript开发了光线追踪算法。
  2. We supported PBR materials and IBL effects. 我们支持PBR材质和IBL效果。
  3. We implemented customized ray tracing on specific regions, such as glass and mirror. 我们实现了面向玻璃或镜子等指定区域的定制化光线追踪。

Ray Tracing on TPU

在神经网络专用处理器(TPU)上实现光线追踪

Duration: Aug. 2022 - Nov. 2022

起止时间：2022.8 - 2022.11

  1. We implemented the ray tracing algorithm using RISC-V C kernels on TPU hardware. 我们在TPU硬件上使用RISC-V的C语言核实现了光线追踪算法。
  2. We implemented ray tracing intersection calculations in hardware, including accelerated structure traversal and triangle intersection. 我们在硬件上实现了光线追踪求交计算，包括加速结构遍历核三角形求交。
  3. We implemented ray tracing shading calculations on hardware, including light source shading and material shading. 我们在硬件上实现了光线追踪着色计算，包括光源着色和材质着色。

Physically Based Optics Ray Tracing Software

基于物理的光线追踪光学仿真软件

Duration: Apr. 2022 - Jul. 2023

起止时间：2022.4-2023.7

  1. We developed a polarized ray tracing engine based on Muller Matrix and Stokes Vector. 我们基于穆勒矩阵和斯托克斯向量研发了一个偏振光线追踪引擎。
  2. We implemented many advanced functions with reference to commercial software，such as measured polarized BRDFs and coating. 我们参考商业软件实现了很多先进的功能，例如测量偏振BRDF材质以及膜层材质。
  3. We supported high-precision simulation and gradient-based optimization for both illumination optics and imaging optics. 我们面向照明光学和成像光学支持了高精度的仿真和基于梯度的优化。
